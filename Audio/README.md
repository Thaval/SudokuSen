# Audio Files

This folder should contain the following audio files for the Sudoku game.

## SFX (Sound Effects)
Place these files in `.wav` or `.ogg` format:

- `click.wav` / `click.ogg` - General button click sound
- `cell_select.wav` / `cell_select.ogg` - Cell selection on the Sudoku grid
- `number_place.wav` / `number_place.ogg` - Placing a correct number
- `number_remove.wav` / `number_remove.ogg` - Removing/erasing a number
- `note_place.wav` / `note_place.ogg` - Adding a note/candidate
- `note_remove.wav` / `note_remove.ogg` - Removing a note/candidate
- `error.wav` / `error.ogg` - Error sound (wrong number)
- `success.wav` / `success.ogg` - Success feedback
- `win.wav` / `win.ogg` - Victory fanfare

## Music
Place these files in `.ogg` or `.mp3` format:

- `menu_music.ogg` / `menu_music.mp3` - Chill background music for the main menu
- `game_music.ogg` / `game_music.mp3` - Relaxing background music during gameplay

## Recommendations

- **SFX**: Keep sounds short (< 1 second) and subtle
- **Music**: Use loopable tracks at moderate volume
- Recommended formats: `.ogg` (better compression) or `.wav` (uncompressed)
- Music volume is default 50%, SFX default 80%
